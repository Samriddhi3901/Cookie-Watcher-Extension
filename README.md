# Cookie Watcher Extension

> Real-time Cookie Monitoring and Management for a Safer Browsing Experience

## About

Cookie Watcher is a powerful next-generation browser extension designed to provide users with unparalleled control and insight into the cookies active during their browsing sessions. With real-time monitoring, detailed analysis, and intuitive management tools, Cookie Watcher empowers you to understand precisely how websites are utilizing cookies, enhance your online privacy, and optimize your browsing performance. It monitors, analyzes, and allows you to manage cookies, helping you maintain a secure and efficient online presence.

## Built With

- Node.js

## Installation

```bash
To install Cookie Watcher, follow these steps:

1. Download the extension package from the GitHub releases page or clone the repository.
2. Open your browser's extension management page (e.g., `chrome://extensions` for Chrome, `about:addons` for Firefox).
3. Enable "Developer mode" (if applicable).
4. Click "Load unpacked" or "Load Temporary Add-on" and select the extension directory.
5. The Cookie Watcher icon should now appear in your browser toolbar.
```

## Usage

```javascript
Once installed, Cookie Watcher automatically begins monitoring cookies in the background. Here's how to get the most out of it:

1. **Click the Cookie Watcher Icon:** Click the extension icon in your browser toolbar to open the main dashboard.
2. **View Active Cookies:** The dashboard provides a real-time list of all cookies currently active on the website you are visiting.
3. **Analyze Cookie Details:** Click on any cookie to view detailed information, including its domain, expiration, value, and security attributes.
4. **Manage Cookies:** From the details view, you can choose to:
    - **Delete:** Remove the individual cookie.
    - **Block Domain:** Prevent all cookies from a specific domain.
    - **Allow Temporarily:** Permit a specific cookie for the current session only.
5. **Global Settings:** Access the extension's settings to configure notification preferences, set default actions for new cookies, and manage your blocked/allowed lists.
```

## Features

- Real-time cookie monitoring and analysis.
- Detailed cookie attribute display (domain, expiry, security flags).
- Intuitive interface for easy cookie management.
- Option to delete individual cookies.
- Ability to block cookies from specific domains.
- Temporary cookie allowance for flexible browsing.
- Customizable notification settings for new cookies.
